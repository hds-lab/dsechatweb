Adium Configuration

Adium is a free IM program for Mac OS X.

Download and install Adium. Then follow the instructions below.

1. Add an account

If you have just installed Adium, you will see the Setup Assistant screen (pictured).

You can also add new accounts under Adium > Preferences in the Accounts tab.

Click Continue to setup your account.

2. Enter your account information

Set the Service to XMPP (Jabber).

The Jabber ID should be [username]@{{ xmpp_server }}.

Enter your chat account password.

Click the Continue button to connect.

3. Join the chat room

In the menus, go to File > Join Group Chat.... The Join Chat window (pictured) will appear.

For Chat Room Name enter {{ xmpp_muc_room }}

For Handle, enter your actual name. This is how other people will see you in the chat room.

Click Join to open the chat room.

4. Save the chat room

With the chat room open, in the menus go to Contact > Add Group Chat Bookmark.

In the box that appears, click Add to save the room in your Contacts.

You are now connected to the chat room.
You can find the room again in your Contacts.

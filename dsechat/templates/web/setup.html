Chat Client Configuration

You can connect to the data science chat room using any client that supports XMPP.

The data science chat room uses XMPP/Jabber. XMPP is an open chat protocol that is supported by a wide range of client programs and services.

We provide instructions for connecting to the data science chat room with Pidgin and Adium, two popular open-source chat programs. However, the setup should be similar if you want to connect with another XMPP client.

Note: Before you can connect, you must have a valid username and password.
